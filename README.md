# Gamer Zone

A React-based gaming zone web app built with **React** and **JavaScript**.  
This project was created as a learning exercise to practice **React fundamentals** and **front-end web development** with a serverless backend.

## 🎯 Purpose
This project was developed to practice:
- React component structure and state management
- JavaScript ES6+ features
- CSS styling and layout
- Building interactive UI elements for games
- Integrating serverless backend functions using Netlify

gamer-zone provides an interactive environment to play simple games while exploring full-stack development with front-end React and backend serverless functions.

***

## 💡 Learning Takeaways
Through this project, I learned to:
- Work with React 16 and 17 versions and their differences
- Manage component lifecycle and state updates
- Use modern JavaScript syntax and tools
- Apply CSS for responsive and engaging UI
- Implement serverless backend logic using Netlify Functions
- Deploy full React apps with backend to Netlify seamlessly

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Ibrahim-Rezq/gamer-zone.git
   ```

2. Navigate to the project directory:
   ```bash
   cd gamer-zone
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open `http://localhost:3000` in your browser to view the app.
